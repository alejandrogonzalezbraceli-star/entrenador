# Plan semanal — Semana del 14 de abril 2026
**Objetivo:** Hipertrofia | **Split:** Push A / Pull A / Legs / Push B / Pull B

> Pesos sugeridos son puntos de partida conservadores (sin 1RM de referencia).
> Ajusta en el primer set según el RIR real: si terminas con RIR >3, sube peso.
> Si terminas con RIR 0 (fallo), baja peso la siguiente serie.

**Notas de la semana:**
- Primera semana: pesos conservadores. El objetivo es aprender el estímulo correcto, no ir al límite.
- Registra el RIR real de cada serie — es el dato más valioso para ajustar la semana siguiente.
- Descanso entre series: 2-3 min en compuestos, 60-90s en aislamientos.
- Si una sesión se alarga más de 80 min, elimina el último ejercicio de aislamiento.
